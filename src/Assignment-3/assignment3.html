Importance of using Iframe

The Iframe tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document.Frames are inline frames that are sometimes referred to as floating frames. One of the main advantages of the iframe tag is that it can be used to load external html or xhtml pages into the current document without distrubing the original page.
